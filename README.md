ResumePro is a state-of-the-art, AI-powered resume building platform designed for the modern professional. It combines beautiful design with powerful AI tools to help you land your dream job.

## ✨ Key Features

- **60+ Premium Templates**: Industrially tailored designs ranging from Minimalist to Creative and Executive.
- **🤖 AI Assistant**: Get real-time, professional suggestions for every section of your resume.
- **✉️ AI Cover Letter Generator**: Generate perfectly matched cover letters in seconds based on your specific resume data.
- **💡 Live Resume Analysis**: Get instant feedback on your resume's strength with our real-time tips widget.
- **☁️ Real-time Cloud Sync**: Your changes are automatically saved to the database as you type.
- **🎨 Custom Design Tools**: Dynamic color pickers, gradient backgrounds, and professional QR codes.
- **🔒 Secure Authentication**: Full user dashboard with secure login and profile management.
- **🛠️ Admin Dashboard**: High-level system overview and user management capabilities.
- **🚀 One-Click Export**: Download high-quality PDF resumes that are ATS-compliant.

## 🛠️ Tech Stack

- **Frontend**: React (Hooks, Context API), Vanilla CSS (Custom Keyframes & Animations).
- **Backend**: Node.js, Express.js.
- **Database**: MongoDB (Mongoose ODM).
- **AI Integration**: Custom prompt engineering for role-specific content.
- **Security**: JWT Authentication, Password Hashing.

## 🏗️ Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/Abhishek-0904/my-project.git
```

### 2. Setup Backend
```bash
cd back-end
npm install
# Create .env file with PORT and MONGO_URI
npm run dev
```

### 3. Setup Frontend
```bash
cd front-end
npm install
npm run dev
```

## 🚀 Deployment

- **Frontend**: Deploy on [Vercel](https://vercel.com/) or [Netlify](https://www.netlify.com/).
- **Backend**: Deploy on [Render](https://render.com/) or [Railway](https://railway.app/).
- **Database**: Use [MongoDB Atlas](https://www.mongodb.com/cloud/atlas) for free tier hosting.
